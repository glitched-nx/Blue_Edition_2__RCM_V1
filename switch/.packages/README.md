# Ultrahand Overlay - Beispiele

Dieses Repository enthält eine Sammlung nützlicher Beispiele, die die Fähigkeiten und die Verwendung des Ultrahand-Overlays demonstrieren.

Ultrahand Overlay ist ein leistungsstarkes Werkzeug, das es Ihnen ermöglicht, die Overlays Ihres Geräts anzupassen und zu verbessern, um die Leistung zu optimieren, die Funktionalität zu verbessern und Ihr Erlebnis zu personalisieren.

## Beispiele

- [Cool Curves](Cool%20Curves/README.md): Erkunden Sie das Beispiel Cool Curves, um zu entdecken, wie Sie die Lüfterkurveneinstellungen durch eine einfache Benutzeroberfläche modifizieren und optimieren können.

- [Host Guard](Host%20Guard/README.md): Das Beispiel Host Guard demonstriert die Nutzung der Overlay-Anpassung zur Verbesserung der Sicherheit und zum Schutz Ihres Geräts vor Servern, auf die Sie möglicherweise nicht zugreifen möchten.

- [Mod Master](Mod%20Master/README.md): Tauchen Sie in das Beispiel Mod Master ein, um zu erfahren, wie Sie das Ultrahand Overlay nutzen können, um Ihre Lieblingsmodifikationen für Spiele zu verwalten und zu optimieren.

- [OC Toolkit](OC%20Toolkit/README.md): OC Toolkit zeigt, wie Sie die Leistung Ihres Geräts an die Grenzen bringen können, indem Sie benutzerdefinierte Übertaktungen, Timings und Unterspannungseinstellungen auf Kip-Dateien anwenden.

- [Broomstick](Broomstick/README.md): Das Beispiel Broomstick bietet eine bequeme Möglichkeit, bestimmte Overlays zu entladen und wiederherzustellen, um Systemressourcen freizugeben und die Leistung zu verbessern.

- [l4t_reboot](l4t_reboot/README.md): Verwenden Sie das Dienstprogramm l4t_reboot unter Linux, um Systemneustarts in Ubuntu einfach durchzuführen.

- [config.ini](https://github.com/ppkantorski/Ultrahand-Overlay/blob/main/examples/config.ini): Grundlegende Systembefehle für Ihr Gerät.

## Erste Schritte

Um diese Beispiele zu verwenden, befolgen Sie diese Schritte:

1. Klonen oder laden Sie dieses Repository auf Ihren lokalen Rechner herunter.
2. Kopieren Sie die Inhalte des gewünschten Beispielordners in `/switch/.packages/` auf Ihrem Gerät.
3. Befolgen Sie die spezifischen Anweisungen, die in der README-Datei jedes Beispiels bereitgestellt sind, um die Overlays zu konfigurieren und anzuwenden.

Hinweis: Bevor Sie Overlays oder Modifikationen anwenden, stellen Sie sicher, dass Sie das potenzielle Auswirkungen auf Ihr Gerät verstehen und die Anweisungen sorgfältig befolgen.
